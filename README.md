# جواهری استقلال — داشبورد قیمت (غیر رسمی)

فایل‌ها:
- `index.html` — صفحهٔ اصلی (ورود + اپ)
- `style.css` — استایل‌ها
- `app.js` — منطق کامل (Rule Engine runtime-decode، fingerprint، DevTools protection)
- `manifest.json` — PWA
- `sw.js` — service worker (اختیاری ولی توصیه‌شده)

نصب ساده:
1. همهٔ فایل‌ها را در یک پوشه قرار بده و آن فولدر را روی GitHub (repo) قرار بده.
2. در GitHub Pages، شاخهٔ `main` یا `gh-pages` را فعال کن — یا فایل‌ها را به صورت محلی باز کن.
3. اولین بار هنگام ورود یک حالت انتخاب کن (Shop یا Full). انتخاب ذخیره می‌شود.
4. برای نصب روی موبایل/دسکتاپ: مرورگر معمولا پیشنهاد می‌دهد یا از منوی مرورگر گزینهٔ Add to Home screen را انتخاب کن.

نکات امنیتی:
- این پروژه **بدون سرور** است؛ همهٔ داده‌ها روی دستگاه کاربر ذخیره می‌شوند.
- لایه‌های بازدارنده (minify-like، runtime-decode، fingerprint، DevTools-detection) برای بازدارندگی طراحی شده‌اند — امنیت صددرصدی را تضمین نمی‌کنند.
- منبع قیمت‌ها در فوتر ذکر شده: `تابان گوهر`.
